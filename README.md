# Données et Script - Mémoire Théorie de l'Autodétermination et PES

Dans une démarche de Science Ouverte, nous mettons à disposition l'intégralité de nos données et de nos traitements statistiques.

## Contenu de ce dépôt
* **`donnees_anonymisees.csv`** : La base de données brute, purgée de toute information nominative. Elle contient les scores des élèves au Temps 1, au Temps 2, ainsi que les mesures de processus (Semaines 1 et 2).
* **`script_analyse_statistique.R`** : Le code complet exécuté sur RStudio. Il comprend le nettoyage des données, le calcul de l'Indice d'Autodétermination (SDI), l'analyse d'attrition, l'évolution longitudinale et les régressions multiples.
